# figo — 品牌资产

> 把设计稿变成可运行的原生 UI：**design → runtime**。

## 核心概念 — Portal F / 转换入口

新标记把三个产品含义压缩进一个几何符号：

- **F**：Figo 的首字母，也是品牌的识别骨架。
- **开放画布**：外框代表 Figma、HTML 和结构化设计稿等输入。
- **向右箭头**：设计跨过转换管线，进入 Figo 运行时或 Godot、Unity、Cocos 等引擎。

符号使用少量粗线条和开放轮廓，在 favicon、CLI、编辑器和视频中都能保持辨识度。字标完全由 SVG 几何轮廓构成，不依赖系统字体。

## 配色

| token | hex | 含义 |
|---|---|---|
| `--brand-mint` | `#23CDB8` | 输入、创造、设计源 |
| `--brand-blue` | `#4B7CFF` | 转换管线、可靠执行 |
| `--brand-violet` | `#7047EB` | 输出、跨引擎能力 |
| `--brand-ink` | `#111827` | 浅色背景字标 |
| `--brand-night` | `#0D1424` | 深色背景 / App 图标 |
| `--brand-paper` | `#F4F7FF` | 深色背景字标 |

主渐变：`linear-gradient(135deg, #23CDB8 0%, #4B7CFF 52%, #7047EB 100%)`。

## 文件

| 文件 | 用途 |
|---|---|
| `figo-mark.svg` | 独立主标记，透明背景；文档、导航栏、合作品牌墙 |
| `figo-app-icon.svg` | 深色圆角底图标；App、favicon、仓库头像、小尺寸场景 |
| `figo-logo.svg` | 浅色背景横版字标 |
| `figo-logo-dark.svg` | 深色背景横版字标 |

## 使用规范

- **最小尺寸**：独立标记不小于 20px；横版字标不小于 96px 宽。
- **安全边距**：四周至少保留标记线宽的 2 倍；不要让文字或边框贴近开放出口。
- **小尺寸**：16–32px 一律使用 `figo-app-icon.svg`，不要使用横版字标。
- **背景**：浅色界面使用 `figo-logo.svg`；深色界面使用 `figo-logo-dark.svg`。
- **不要修改**：不得旋转标记、关闭右侧出口、反转箭头方向或单独改变渐变顺序。
- **单色场景**：整个标记统一使用白色、`#111827` 或 `#4B7CFF`，保留开放轮廓。
